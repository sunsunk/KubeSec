If the `source` is `none`, DevSpace will use the `default` value as value for this config variable.
```yaml {3,5-7}
images:
  database:
    image: "mysql:${MYSQL_VERSION}"
vars:
- name: MYSQL_VERSION
  source: none
  default: "5.5"
```
