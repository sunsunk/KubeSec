Welcome to the in-toto documentation!
=====================================

in-toto is a framework for software supply chain security. This documentation
covers the `in-toto Python reference implementation
<https://github.com/in-toto/in-toto>`_ of the `in-toto specification
<https://github.com/in-toto/docs>`_. More information about in-toto can be
found at `in-toto.io <https://in-toto.io>`_.
